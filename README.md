# Africa Pulse

Dashboard de visualisation de données africaines — population, PIB et accès internet pour 15 pays, avec filtres régionaux, graphiques animés et fiches détail par pays.

🔗 **Demo live** : [africa-pulse.netlify.app](https://africa-pulse.netlify.app)

## Stack
React · Recharts · Vite · Netlify

## Fonctionnalités
- Filtres par région (Ouest, Est, Nord, Centrale, Australe)
- Graphiques animés : PIB, démographie, accès internet
- Fiche détail par pays au clic
- Compteurs animés au chargement
- Responsive mobile

## Lancer en local
```bash
npm install
npm run dev
```

## Pourquoi ce projet ?
L'Afrique est souvent absente des outils de dataviz accessibles. Africa Pulse rend la donnée africaine visible et exploitable — pour les étudiants, chercheurs, journalistes ou décideurs.
